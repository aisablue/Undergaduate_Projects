# Scrabble 2021 (Greek Edition)

A text-based version of the classic **Scrabble** game written in Python.  
Developed by **A. Iliopoulou (A.E.M. 14869)** as part of a university project.

---

##  Description
This program simulates a game of Scrabble in **Greek**, where a human player competes against the computer.  
It includes:
- Word validation using a Greek dictionary (`greek7.txt`)
- Scoring based on letter values
- JSON-based score saving
- A semi-random computer algorithm ("Smart-Fail") that plays in a human-like way

---

##  About the Code
All **comments, documentation, and variable names** inside the source files are **kept in Greek** on purpose.  
This was done to **preserve the originality and authenticity** of the original project exactly as it was developed in 2021.
